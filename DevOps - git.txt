git commands::
================================================
intialize git for the first time:
1.install git for windows
2. create a master folder
3. goto that folder and gitbash here
git init
git remote add origin "<<<Repository URL>>"

5. add the files from remote repository to the local repository
git pull origin master 

5. git add -A // adding all the new files to repo
6. git commit -a // commiting all the files to repo
